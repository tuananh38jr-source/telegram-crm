Huong dan chi tiet cac cach login

Cach 1: Login tren CRM (de nhat — khuyen nghi)

  1. Nhan nut "LOGIN TREN CRM" o tren
  2. Chon ten account CRM + nhap so dien thoai Telegram
  3. Nhan "Dang nhap" → server tu mo ads.telegram.org va nhap so DT
  4. Mo app Telegram tren dien thoai → bam xac nhan dang nhap
  5. Doi vai giay → he thong tu luu cookies va bao thanh cong

Server tu dong mo trinh duyet, nhap so DT, luu tat ca cookies (ke ca httpOnly).

Thêm Account Telegram Ads

Upload Cookie File (sau khi login local)

Cookie login workflow:

  1. Chạy login local trên máy có trình duyệt:
    python telegram_ads_auto_export.py --login --account "Tên"
  2. Script mở browser → đăng nhập Telegram → lưu cookie vào telegram_ads_cookies/
  3. Upload file .json cookie ở form trên để CRM sử dụng cho auto-export
  4. Sau khi upload, nhấn "Auto Export" để CRM chạy export tự động bằng cookie đã lưu

Cách sử dụng:

  1. Tạo account ở form bên phải
  2. Vào ads.telegram.org → Statistics → export 2 file CSV:
    • File Views: chọn "Days" → Export
    • File Budget: chọn "Days" + tick "Spent budget" → Export
  3. Upload cả 2 files vào account tương ứng
  4. Hệ thống tự merge theo ngày và hiển thị tổng hợp

Format CSV thực tế:

  • Tab-separated (không phải comma)
  • Ngày: "30 Dec 2025"
  • Số thập phân: "23,3973" (= 23.3973)
  • File budget có chữ "budget" trong tên
